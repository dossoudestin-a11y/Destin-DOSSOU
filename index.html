Métiers de l'informatique

Développeur Web/App

Crée sites, apps, jeux. HTML, JS, Python. C’est le métier de base pour débuter.

Cybersécurité

Protège les données, les banques, les gouvernements contre les hackers. Très demandé.

Data/IA

Analyse des millions de données. Entraîne ChatGPT, TikTok, Netflix. Le futur.

DevOps/Cloud

Fait tourner les serveurs d’AWS, Google. S’assure que tout marche 24h/24.

UX/UI Design

Rend les apps belles et faciles à utiliser. Moitié art, moitié tech.

Support IT

Dépanne les PC, réseaux, imprimantes. Porte d’entrée idéale pour débuter.
